# Assets nécessaires

Pour que l'application fonctionne correctement, vous devez créer les fichiers suivants dans le dossier `apps/mobile/assets/` :

## Fichiers requis

1. **icon.png** (1024x1024px)
   - Icône de l'application pour iOS et Android
   - Format PNG avec transparence

2. **splash.png** (1284x2778px recommandé)
   - Image de splash screen
   - Format PNG

3. **adaptive-icon.png** (1024x1024px)
   - Icône adaptative pour Android
   - Format PNG avec transparence

4. **favicon.png** (48x48px)
   - Favicon pour la version web
   - Format PNG

## Génération des assets

Vous pouvez utiliser des outils en ligne comme :
- [App Icon Generator](https://www.appicon.co/)
- [Expo Asset Generator](https://docs.expo.dev/guides/app-icons/)

Ou créer manuellement les fichiers avec les dimensions spécifiées.
